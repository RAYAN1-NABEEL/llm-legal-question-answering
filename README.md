# LLM Legal Question Answering

This project fine-tunes a Large Language Model to answer legal questions using domain-specific data.

## Model
Mistral-7B-Instruct

## Techniques
- LoRA fine-tuning
- 4-bit quantization
- HuggingFace Transformers

## Goal
Build a legal assistant using LLM fine-tuning.
